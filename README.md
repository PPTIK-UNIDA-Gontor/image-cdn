Ikutin step dimari biar kagak nyasar:

1.  **Upload Dulu Ngab** 
    Pertama, lau **cabs** ke folder yang sesuai (atau bikin folder baru biar rapi, jangan berantakan kek kapal pecah). Terus upload dah ntuh file gambar lau di situ. lau juga sabeb ntuh buat clone ini repo.

2.  **Cuss Buka Filenya**
